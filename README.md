## Dev Setup

creating virtual environment and installing dependencies.

```powershell
$ mkdir .venv
$ pipenv install --dev
```

running the app:

```powershell
$ pipenv shell
$ python manage.py runserver
```

<br>

### Normal Setup

```powershell
$ mkdir .venv
$ pipenv install
$ pipenv run python manage.py runserver
```
